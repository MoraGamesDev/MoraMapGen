    Class PropRotationSettingsSO

    ScriptableObject asset defining rotation variance for prop placement.

    public class PropRotationSettingsSO : ScriptableObject
    Remarks

    Configure how props can be randomly rotated when placed, including min/max ranges per axis, incremental snapping, and random deviation. Assign to PropDefinitions or PropOptions to control random rotation. Create via Assets > Create > MoraMapGen > Props > Prop Rotation Settings.
