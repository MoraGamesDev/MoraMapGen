    TryGetPath(BakedMap, Vector2Int, Vector2Int, out SearchTile, params EnvType[])

    Attempts to find a path between two positions in the baked map, using a simplified A* search over valid environment types.

    Declaration
    public static bool TryGetPath(this BakedMap bakedMap, Vector2Int start, Vector2Int end, out SearchTile pathEnd, params EnvType[] validTypes)
    Parameters
    Type Name Description
    BakedMap bakedMap

    The map to search through.
    Vector2Int start

    The starting grid position.
    Vector2Int end

    The target grid position.
    SearchTile pathEnd

    If the path is found, this will contain the final SearchTile in the chain, from which the full path can be reconstructed using GetPathPositions().
    EnvType[] validTypes

    Optional list of valid environment types (e.g., Room, Corridor) that the path may traverse.
    Returns
    Type Description
    bool

    True if a valid path was found; otherwise, false.
    Examples

    Vector2Int start = map.GetGridSpace(startObj.position).GridPosition; Vector2Int end = map.GetGridSpace(endObj.position).GridPosition;

    if (map.TryGetPath(start, end, out var pathEnd, EnvType.Room, EnvType.Corridor)) { foreach (var pos in pathEnd.GetPathPositions()) { Vector3 worldPos = map.GetWorldPosition(pos); Instantiate(markerPrefab, worldPos, Quaternion.identity); } } else { Debug.Log("No path found."); }
