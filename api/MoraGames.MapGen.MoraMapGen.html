    Class MoraMapGen

    Main MonoBehaviour component for generating procedural grid-based dungeon maps.

    Inheritance
    object
    MoraMapGen
    Namespace: MoraGames.MapGen
    Assembly: MoraMapGen.Runtime.dll
    Syntax
    public class MoraMapGen : MonoBehaviour
    Remarks

    Attach this component to a GameObject to generate procedural dungeon maps at runtime or in the editor. Maps consist of rooms and corridors arranged on a grid, with automatic prop placement and navmesh generation.
