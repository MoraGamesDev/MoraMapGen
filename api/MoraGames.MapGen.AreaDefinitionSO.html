    Class AreaDefinitionSO

    ScriptableObject asset that defines the name, tile modules, and props and their placement rules for a map area.

    Inheritance

    Namespace: MoraGames.MapGen
    Assembly: MoraMapGen.Runtime.dll

    Remarks

    Create area definitions for general types of rooms and corridors (e.g: "Castle Hall", or "Technical Corridor"), as well as specific named ones (e.g: "Throne Room", or "Command Bridge"). Create via Assets > Create > MoraMapGen > Area Definition.
