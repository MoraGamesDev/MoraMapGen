    Enum PropPlacementType

    Flags enum defining where props can be placed in the generated map.

    Namespace: MoraGames.MapGen.Advanced.Props
    Assembly: MoraMapGen.Runtime.dll
    Syntax
    [Flags]
public enum PropPlacementType
    Remarks

    Multiple flags can be combined to allow a prop to spawn at different types of locations. Use bitwise operations to check for specific placement types.

    Fields

    Name Description
    None

    No placement type specified.
    OnCorridorCorners

    Place on corridor corner tiles.
    OnCorridorDoorSpaces

    Place on corridor tiles containing doors.
    OnCorridorIntersections

    Place on corridor intersection tiles (3-way "T" and 4-way X intersections).
    OnCorridorStraights

    Place on straight corridor tiles.
    OnDeadEnds

    Place on dead-end tiles.
    OnPropSpot

    Place on manually positioned PropSpot components in tile modules.
    OnRoomCenters

    Place on room centroid.
    OnRoomCorners

    Place on room corner tiles.
    OnRoomDoorSpaces

    Place on room tiles containing doors.
