    Class PropSpot

    MonoBehaviour component that marks a specific position in a tile module where props can be placed.

    Inheritance
    object
    PropSpot
    Namespace: MoraGames.MapGen.Props
    Assembly: MoraMapGen.Runtime.dll

    public class PropSpot : MonoBehaviour
    Remarks

    Place this component in your tile module prefabs to define prop spawn points. Configure placement variance and assign a PropChannel to control which props spawn here. Note that the PropPlacement component only looks for these in module prefabs spawned as part of the map generation process. If you manually place them anywhere else you should create a custom IMapGenPostProcess that eithe processes them in a different way, or makes them children of tile objects in the Before Props phase.
