    Class ProceduralPropSpotOffsetsSO

    ScriptableObject asset defining position variance for Procedural Prop Spots.

    Inheritance

    ProceduralPropSpotOffsetsSO
    Namespace: MoraGames.MapGen.Props
    Assembly: MoraMapGen.Runtime.dll
    Syntax
    public class ProceduralPropSpotOffsetsSO : ScriptableObject
    Remarks

    Configure different position offset ranges for each type of procedural placement location (dead-ends, corners, intersections, etc.). Assign to PropPlacement component or override per PropDefinition in your Prop Categories. Create via Assets > Create > MoraMapGen > Props > Procedural Prop Spot Offsets.
