    Class PropChannelSO

    ScriptableObject asset used as a channel to assign prop definitions and prop spots.

    Remarks

    Create multiple PropChannel assets to indicate different placements (e.g., "Wall Center", "Wall Floor", "Floor", etc). Assign channels to both PropSpots in tile modules and PropDefinitions to control where specific props can spawn. Channels also allow per-area density multipliers in AreaDefinitionSO. Create via Assets > Create > MoraMapGen > Props > Prop Channel.
