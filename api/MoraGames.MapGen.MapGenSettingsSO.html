    Class MapGenSettingsSO

    ScriptableObject asset defining all parameters for procedural map generation.

    Inheritance

    Assembly: MoraMapGen.Runtime.dll
    Syntax
    public class MapGenSettingsSO : ScriptableObject
    Remarks

    Configure map size, room creation mode, corridor design, randomness, and other generation parameters through this asset. Create via Assets > Create > MoraMapGen > Map Gen Settings Asset.
