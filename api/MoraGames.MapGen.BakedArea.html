    Class BakedArea

    Represents a contiguous group of tiles forming either a room or corridor in the generated map.

    Remarks

    BakedArea provides access to the grid spaces, prop spots, and connection information for a room or corridor. Use this class to query area properties, place props, and understand how areas connect to each other.
