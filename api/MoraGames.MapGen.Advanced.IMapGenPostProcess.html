    Interface IMapGenPostProcess

    Interface for components that perform post-processing operations after map generation.

    Namespace: MoraGames.MapGen.Advanced
    Assembly: MoraMapGen.Runtime.dll
    Syntax
    public interface IMapGenPostProcess
    Remarks

    Implement this interface to add custom logic that runs after the map is baked, such as enemy spawning, lighting setup, or custom prop placement. Attach the implementing component to the same GameObject as your MoraMapGen component.
