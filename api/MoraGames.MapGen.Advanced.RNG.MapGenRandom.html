    Provides deterministic random number generation utilities for map generation and prop placement.

    Namespace: MoraGames.MapGen.Advanced.RNG
    Assembly: MoraMapGen.Runtime.dll
    Syntax
    public static class MapGenRandom
    Remarks

    This static class offers various distribution modes (uniform, normal, biased) and convenience methods for common randomization tasks. All methods use seeded RNGs for deterministic, reproducible results. Initialize with Init(seed) before first use, or it will auto-initialize with seed 0.

    NormalRange(Random, int, int, float, int)

    Generates an integer sampled from a normal (Gaussian) distribution within a given range.

    Declaration
    public static int NormalRange(Random rng, int minInclusive, int maxExclusive, float spreadFactor = 0.2, int maxRerolls = 10)
    Parameters
    Type Name Description
    Random rng

    The random number generator to use.
    int minInclusive

    The minimum inclusive bound of the range.
    int maxExclusive

    The maximum exclusive bound of the range.
    float spreadFactor

    Controls the spread of the distribution relative to the range.
    A smaller value makes results cluster tightly around the mean,
    while a larger value increases the likelihood of results near the extremes.
    int maxRerolls

    The maximum number of times to resample if the generated value falls outside the range.
    After this limit is reached, the result is clamped into the range.
    Returns
    Type Description
    int

    An integer sampled from a normal distribution centered on the midpoint of the range,
    constrained to be equal to or greater than minInclusive and up to but not incuding maxExclusive.
    Examples

    Example usage:

    var rng = new System.Random();

// Produces numbers mostly clustered around 50
int tightCluster = Normal(rng, 0, 100, spreadFactor: 0.2f);

// Produces numbers more evenly spread, often near the extremes
int wideSpread = Normal(rng, 0, 100, spreadFactor: 1.0f);

// Faster but potentially less accurate (limits rerolls)
int fastSample = Normal(rng, 0, 100, spreadFactor: 0.5f, maxRerolls: 2);

    NormalValue(Random, float, float)

    Generates a random float sampled from a normal (Gaussian) distribution using the Box-Muller transform.

    Declaration
    public static float NormalValue(Random rng, float mean = 0, float stdDev = 1)
    Parameters
    Type Name Description
    Random rng

    The random number generator to use.
    float mean

    The mean (average) value of the distribution.
    Default is 0.
    float stdDev

    The standard deviation of the distribution, controlling the spread of values.
    Default is 1.
    Returns
    Type Description
    float

    A random float sampled from a normal distribution with the specified mean and stdDev.
    Remarks

    This method uses the Box-Muller transform to convert two uniformly distributed random numbers into a normally distributed sample.
    Results are unbounded, so values can lie arbitrarily far from the mean.

    Examples

    Example usage:

    var rng = new System.Random();

// Sample from a standard normal distribution (mean=0, stdDev=1)
float z = Normal(rng);

// Sample heights around 170 cm with ~10 cm variation
float height = Normal(rng, mean: 170f, stdDev: 10f);
