    Class PropPlacement

    Component that handles prop placement as a post-processing step after map generation.

    Inheritance
    object

    IMapGenPostProcess
    Namespace: MoraGames.MapGen.Props

    Syntax
    public class PropPlacement : MonoBehaviour, IMapGenPostProcess
    Remarks

    Attach this component to the same GameObject as your MoraMapGen component to automatically place props based on the Prop Categories in your Area Definitions and Tile Module Definitions. Handles required props, optional props, and always-spawn props with collision checking and variance.
