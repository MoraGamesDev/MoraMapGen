    Class BakedMap

    Represents the finalized, runtime-accessible structure of a generated map.

    Syntax
    public class BakedMap
    Remarks

    BakedMap provides a high-level interface for querying rooms, corridors, grid spaces, and navigational data after map generation completes. Use this class to interact with the generated map at runtime for gameplay logic, AI navigation, and prop placement.
