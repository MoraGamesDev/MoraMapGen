    Room Shapes

    Room Shape assets define the shape of rooms that can be generated when using the Chunks or Hybrid Room Creation Mode in the Settings.

    Creating a Room Shape asset

    To create a new Room Shape asset, right-click in your Project Assets and select "Create/MoraMapGen/Room Shape".
