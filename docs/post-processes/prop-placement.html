    Prop Placement

    As an option, you can have props get placed to fill your map. Props can be defined on a per-Area Definition basis, and/or on a per-Tile Module Definition basis. For example, you can have props specific to a “Kitchen” Area Definition which won’t appear in other rooms or corridors, and you can also have props assigned to a “Stone Walls” Tile Module Definition so that they will appear in any rooms or corridors which use that Tile Module Definition asset.

    Steps for setting up Prop Placement

    To have props spawn in your map, you must:

    • Setup the Prop Placement component
    • Create Prop Channels
    • Create prop prefabs
    • Add Prop Spots to your module prefabs
    • Define prop categories
